## Що дивитися

- [ ] Немає горизонтального скролу.
- [ ] Верхнє меню не ламається.
- [ ] Адмін-меню не перекриває екран.
- [ ] Картки промокодів читаються.
- [ ] Кнопка копіювання коду зручна.
- [ ] Форми не виходять за екран.
- [ ] Поля вводу не занадто малі.
- [ ] Кнопки не злипаються.
- [ ] Текст не обрізається.
- [ ] Футер не займає половину екрану.
- [ ] Сторінка промокоду зручна на телефоні.
- [ ] Коментарі зручні.
- [ ] Репорт зручний.
- [ ] Збереження промокоду зручне.
- [ ] Guest prompt виглядає нормально.
- [ ] `/admin` хоча б придатна для швидкої модерації з телефона.

## Пріоритетні проблеми для виправлення

1. Горизонтальний скрол.
2. Зламане меню.
3. Неклікабельні кнопки.
4. Форми, які неможливо заповнити з телефона.
5. Сторінка промокоду, де важко скопіювати код.
6. Адмінка, де неможливо схвалити/відхилити код.
