# Юридичний чек-лист ПромоПтахи

Дата: 16.07.2026

Це робочий чек-лист, не юридична консультація.

## Обовʼязково до запуску

- [x] Сторінка `/rules`
- [x] Сторінка `/privacy`
- [x] Сторінка `/terms`
- [x] Дисклеймер: ПромоПтаха не є продавцем
- [x] Дисклеймер: промокоди додаються користувачами
- [x] Дисклеймер: умови перевіряти на сайті магазину
- [x] Правила користувацького контенту
- [x] Правила модерації
- [x] Політика щодо персональних даних
- [x] Контакт для звернень

## Бренди і магазини

- [ ] Не писати “офіційний партнер”, якщо партнерства немає.
- [ ] Не робити сторінки магазинів схожими на офіційний сайт магазину.
- [ ] Обережно з логотипами.
- [ ] Додати дисклеймер, що торговельні марки належать їх власникам.
- [ ] За скаргою правовласника швидко прибирати спірний матеріал.

## Промокоди і реклама

- [ ] Не гарантувати роботу коду.
- [ ] Показувати строк дії, якщо відомий.
- [ ] Показувати джерело, якщо відоме.
- [ ] Дати користувачам можливість скаржитись.
- [ ] Дати користувачам можливість голосувати “не працює”.

## Персональні дані

- [ ] Не показувати email публічно.
- [ ] Дати користувачу канал для запиту на видалення даних.
- [ ] Не збирати зайві персональні дані.
- [ ] Тримати `.env.local` поза Git.
- [ ] Не використовувати service_role key у браузері.

## Бажано після запуску

- [ ] Додати окрему процедуру видалення акаунта.
- [ ] Додати сторінку “Правовласникам”.
- [ ] Додати форму скарги правовласника.
- [ ] Перевірити тексти юристом, якщо сайт почне рости.
